## Project Description
This project analyzes Git as an open-source software and demonstrates Linux shell scripting concepts.

---

## Scripts Overview

### 1. System Identity Report
Displays system info like kernel, user, uptime.

### 2. Package Inspector
Checks if Git is installed and shows its details.

### 3. Disk Auditor
Shows directory size and permissions.

### 4. Log Analyzer
Counts occurrences of keywords in logs.

### 5. Manifesto Generator
Creates a personal open-source philosophy file.
